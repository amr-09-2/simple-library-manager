# Simple Library Manager


  
## Description

So I've been working on this small project called Simple Library Manager. It's built with Python, which I'm still learning. I started it just to practice and see if I could build something real, and I'm pretty happy with how it turned out so far.

## Features
* Adding books to my library
* Keeping a wish list of books I want to read someday
* Moving books from the wish list to the library once I get them
* Donating books I'm done with

## How to run
Make sure Python is installed first, then run:
python library.py
